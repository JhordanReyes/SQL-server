Tutorial - SQL server

Crear una base de datos:

create database Salon

Usar una base de datos:

use Salon

Crear una tabla:

// Importante //

primary key => Llave primaria, solo debe haber una por tabla.
not null => no debe faltar ingresar el dato.

varchar(8) => permite ingresar Cadena de caractered, entre los parentesis () se pone el rango de caracteres.
integer => permite ingresar numeros.
float(8) => permite ingresar numeros decimales.

//Procedemos a crear la tabla//

create table usuarios(
cod_usur varchar(8) primary key not null,
name_usur varchar(10) not null,
dir_usur varchar(20) not null
)

Muestrar caracteristicas de una tabla:

exec sp_columns usuarios

Borrar una base de datos y tabla:

drop database Salon

drop table usuarios

Insertar una fila en la tabla:

insert into usuarios(cod_usur,name_usur,dir_usur)
values("12ab","Jhordan","Jose Galvez"),
("34cd","Joel","Chorrillos"),
("56ef","Luis","Lurin")

Seleccionar una tabla:

* = > selecciona todo
select * from usuarios
